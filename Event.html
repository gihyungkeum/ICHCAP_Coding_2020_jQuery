event 란?

  • 클릭, 마우스이동, 타이핑 등 브라우져에서 일어나는 사건
  • 이벤트 발생시 작동할 로직을 시스템에 알려주면 이벤트 발생시 시스템이 해당 로직을 호출
  • jquery 이벤트는 크로스브라우징을 해결
  • bind로 이벤트 핸들러 설치하고, unbind로 제거
  • trigger로 이벤트 핸들러 강제로 실행
  • click, ready와 같은 이벤트 헬퍼 제공
  • live 사용시 현재 존재 하지 않는 엘리먼트에 이벤트핸들러 설치할 수 있음

bind, unbind, ready, trigger를 이용한 이벤트 설치, 제거, 호출--> on, off, triger로 대체

# 생코에서 사용된 bind, unbind, live, die, reay 메소드는 jquery 1.7 버전이후 일괄적으로 on, off로 대체..>jquery api on click 참조

# 또한 function clickHandler(e)....에서 e는 event를 의미하며 관습적으로 쓰기도 하지만, function clickHandler()와 같이 없어도 무방

function clickHandler(e){
alert('thank you');
}
$(document).bind('ready', function(){
$('#click_me').bind('click', clickHandler);
$('#remove_event').bind('click', function(e){
$('#click_me').unbind('click', clickHandler);
});
$('#trigger_event').bind('click', function(e){
$('#click_me').trigger('click');
});
})
